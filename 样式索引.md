# FundMgmt系统样式索引表（常用控件）

| 名称 | 类名 | 依赖 | 备注 |
| ------ | ------ | ------ | ------ |
| 按钮 | button |  theme.css  | 一般不再设置`宽度` |

``` css
.button
{
    border-style:none;
    color: #B7AFA5;
    cursor:pointer;
    font-size:13px;
    background-color:#5C554D;
    height:26px;
    min-width: 70px;
} 
```
| 名称 | 类名 | 依赖 | 备注 |

``` css
} 
```
