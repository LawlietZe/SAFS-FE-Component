### 产品：
- safs.css
- safs.js
- 视觉设计文档

### 特色：
- 完整的API文档
 - 包含所有开发金融管理系统常用控件以及代码示例。
- 跨终端设计

- 零学习成本
 - 傻瓜式使用说明文档，复制粘贴就能写出适合产品需求的页面。
